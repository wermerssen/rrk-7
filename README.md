# RRK‑7.0 — Recursive‑Emergent Meta‑Structure

This repository contains the complete web presentation of the RRK‑7 framework.  
It includes the bilingual documentation (English/German), the theoretical core components, structural models, figures, and all associated scientific artifacts.  
The project provides a consolidated reference for the RRK ontology, the evolutionary development of the framework, the meta‑structural definitions, and the integration of OSF materials.

## Structure
- `/rrk/` — German documentation  
- `/en_rrk/` — English documentation  
- `/assets/` — logo, stylesheets, figures
